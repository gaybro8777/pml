Best Practice Replacements

pmlp:hasMember

pmlp:hasMember was used to include an pmlp:Agent as part of a pmlp:Organization.

Instead of using pmlp:hasMember, use foaf:member (documentation) instead. Domain is foaf:Group, which is not foaf:Organization.
